## 4 Poin dalam membangun kebiasaan
4 poin berikut saya dapat ketika menonton youtube untuk membangun kebiasaan:
1. Make it obvious (jelas)
2. Make it easy (mudah)
3. Make it attractive (menarik)
4. Make it satisfying (memuaskan)

Jadi saya mencoba untuk membuat kegiatan saya jadi lebih menarik.

Kebetulan saya pernah bikin e-wallet dengan nama tabungin-aja tapi project tersebut mangkrak di localhost. Jadi saya berpikir untuk menulis ulang project tersebut untuk melakukan pomodoro menjadi lebih menarik dan *satisfying*.

## Hadiah uang jika pomodoro selesai
Dengan melakukan satu pomodoro, saya akan mendapatkan 700 rupiah. Dengan target harian 18 pomodoro maka saya akan menghasilkan 12.6k rupiah. Perkiraan dalam satu bulan saya mendapatkan 378k rupiah jika konsisten.  

Akhir bulan dengan 378k, saya boleh menggunakan uang itu untuk apapun yang saya mau, sebagai penghargaan sudah konsisten melakukan pomodoro.
